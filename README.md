# Pure Mitten Junk Removal Website

Static multi-page website for Pure Mitten Junk Removal.

## Pages

- `index.html` - Home
- `services.html` - Services
- `pricing.html` - Load sizes and minimum pickup
- `about.html` - About and service area

## Assets

The site uses files in the `assets/` folder for the logo and Michigan outline.

When uploading to a host or GitHub, keep the folder structure the same so the image paths continue to work.
